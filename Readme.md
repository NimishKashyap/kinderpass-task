# Kinderpass Task Submission

### Setup for Frontend

```
cd frontend
npm i/yarn
```

### Setup for backend

_NOTE:_ Docker and docker-compose have to be installed. Using lastest mongo image

```
// To start mongo container
docker-compose up

cd backend
npm i/yarn
```

### Tech-stack used

Backend:

1. ExpressJS + NodeJS
2. MongoDB in Docker

Frontend:

1. ReactJS
2. React-Bootstrap (UI)
3. Axios
